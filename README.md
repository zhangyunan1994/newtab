# WTab - 自定义新标签页扩展

一个功能丰富、美观的浏览器新标签页扩展，支持搜索引擎切换、背景自定义和书签管理。

## 功能特性

### 🔍 多搜索引擎支持
- 支持 Google、Bing、百度、DuckDuckGo、Yahoo 等多个搜索引擎
- 可快速切换搜索引擎
- 支持设置默认搜索引擎
- 智能识别URL，直接访问网站

### 🎨 背景自定义
- **纯色背景**：选择任意颜色作为背景
- **图片背景**：上传自定义图片作为背景
- **渐变背景**：使用双色渐变创建美观背景
- 所有设置自动保存

### 📚 书签管理
- 快速添加书签
- 美观的网格布局展示
- 自动获取网站图标（favicon）
- 一键删除书签
- 点击书签快速访问

## 安装方法

### Chrome/Edge 浏览器

1. 下载或克隆本项目到本地
2. 打开浏览器，进入扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 开启"开发者模式"（右上角开关）
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 完成！新标签页将自动使用 WTab

## 使用说明

### 搜索功能
- 在搜索框中输入关键词，按回车或点击搜索按钮
- 使用下拉菜单切换搜索引擎
- 直接输入URL或域名可快速访问网站

### 设置背景
1. 点击右上角的设置按钮（⚙️）
2. 在设置面板中选择背景类型：
   - **纯色背景**：选择颜色并应用
   - **图片背景**：点击"上传图片"按钮选择图片
   - **渐变背景**：选择两种颜色创建渐变
3. 设置会自动保存

### 管理书签
1. 点击书签区域的"+"按钮
2. 输入书签名称和URL
3. 点击"保存"完成添加
4. 将鼠标悬停在书签上，点击右上角的"×"可删除书签

### 设置默认搜索引擎
1. 打开设置面板
2. 在"默认搜索引擎"下拉菜单中选择
3. 下次打开新标签页时将使用该搜索引擎

## 注意事项

### 图标文件
由于图标文件需要实际图片，请自行准备以下尺寸的图标：
- `icons/icon16.png` - 16x16 像素
- `icons/icon48.png` - 48x48 像素
- `icons/icon128.png` - 128x128 像素

如果没有图标文件，扩展仍然可以正常工作，只是在扩展管理页面中可能显示默认图标。

### 数据存储
所有设置和书签数据存储在浏览器的同步存储中（`chrome.storage.sync`），可以在不同设备间同步（如果已登录Chrome账户）。
